Project #6 Data Portraits

Description

I have recorded my life status for a week and put it into this table. This is a chart about the relationship between life stress, bubble tea and eating.

Design Process

I put the recorded values into the array and then use if else to determine the color of each circle. Finally, I add the interaction of the weekdays by moving the mouse over.

Reflection

GIORGIA LUPI and STEFANIE POSAVEC's dear data project made me realize that data can have more possibilities. After I finished this project with array, I think I can add more elements and complex data into it.
