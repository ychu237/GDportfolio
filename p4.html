Project #3 Face Generator

Description

In this project, I made a experimental cd player with if else statment and play with time.

Design Process

In the beginning, I drew a sketch in p5js and then started to think about how to make the cd zoom in and out. I also tried to cooporate millis and rotate speed together.

Reflection

This project help me to get familiar with the if else statement. I have difficulties when I want to put time code in it. Overall, it's a challenging but interesting project.
