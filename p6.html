Project #6 Optical Illusion

Description

This is a pearl milk tea code. When the mouse moves over the circle, the circle becomes darker and the pearl disappears when the mouse is clicked.

Design Process

I drew the bubble by using for loop and corporate if else with mouseX, mouse Y to change the color of each pearl. The difficult part is the click. I search some tutorial to write the disappear part.

Reflection

In this project, I learned how to use those functions together and built something that in my mind.
